Reflections:

The objective here is to reflect on what I learned this semester. Well, I've learned how to cobble together some basic html along with some nifty academic factoids regarding the same. This class felt a bit overly weighted on researching these facts verses spending more time doing coding, but it is my first college level class in twenty years. Since then and up until now it was mostly mini boot camps as my way to learn something, but that is the quick and dirty approach for an end to a means usually in the moment of introducing or having been introduced to something new.

Overall, I truly enjoyed this class and feel I was given a solid baseline to move on to more advanced classes in web development if that is the route I choose to go.
